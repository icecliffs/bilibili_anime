### 特点

- 显示番剧
- 显示番剧基本信息

### 使用方法

1. 将本仓库 git 到你网站/wp-content/theme/你的主题/pages/下
2. 打开 bilibili.php 修改 uid

     ![](/Images/2.png)

3. 新建一个页面，然后右下角模板选择【哔哩哔哩追番页面】
   
     ![](/Images/4.png)

4. Enjoying...:)
